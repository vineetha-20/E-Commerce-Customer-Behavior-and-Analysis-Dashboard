E-Commerce Customer Behavior and Analysis
Project Overview

This project focuses on analyzing e-commerce sales and customer behavior using an interactive dashboard built in Microsoft Power BI. The goal is to transform raw transactional data into meaningful insights that help understand business performance, customer purchasing patterns, and product profitability.

The dashboard provides a clear view of important business metrics such as Total Sales, Profit, Shipping Cost, Orders, Customer Segments, Product Categories, and Regional Sales Distribution.

This project was developed as part of the AICTE Microsoft Elevate Internship Program to demonstrate practical skills in data visualization, business intelligence, and data-driven decision making.
